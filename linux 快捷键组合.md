# linux 快捷键组合

## 移动光标

``` 
ctrl+a: 光标跳到行首。
ctrl+e: 光标移到行尾。

ctrl+b: 光标左移一个字符。键盘左
ctrl+f: 光标右移一个字符。键盘右

ctrl+左右键: 往左/右移动一个单词
alt+b: 往左移动一个单词
alt+f: 往右移动一个单词
esc+b ：往左移动一个单词
esc+f ：往右移动一个单词

ctrl+xx ：在命令行单词光标位置和单词首之间移动，或在命令行光标位置和行首之间移动，看系统版本

```



## 编辑命令

```
ctrl+h: 删除光标前一个字符，同backspace 键相同。（有剪切板的功能）
ctrl+d: 删除光标所在字符。（有剪切板的功能）。

ctrl+u: 清除光标前至行首间的所有内容。（有剪切板的功能）
ctrl+k: 清除光标后至行尾的内容。（有剪切板的功能）

ctrl+w: 删除光标前的单词。（有剪切板的功能）
alt+d: 删除光标后的单词。（有剪切板的功能，多个alt操作一起合并到剪切板的功能）
esc+d: 删除光标后的单词
alt+backsapce: 删除当前光标前的单词。（有剪切板的功能，多个alt操作一起合并到剪切板的功能）

ctrl+y: 粘贴剪切板上的内容。将刚才删除的字符粘出来，可认为是进行剪切后的粘贴
ctrl+t: 交换光标位置前的两个字符

alt+.: 使用上一条命令的最后一个参数。
ctrl+- ：回复之前的状态。撤销操作。

esc+c 使下一个单词首字母大写, 同时光标前进一个单词, 如光标停留在单词的某个字母上, 如word中的o字母上, 则o字母变大写. 而不是w
esc+u 使下一个单词所有字母变大写, 同时光标前进一个单词, 同上, 如光标在o字母上, 则ord变大写, w不变.
esc+l 同ESC-U, 但使之全变为小写.
```



## 查找历史命令

```
ctrl+p: 显示当前命令的上一条历史命令
ctrl+n: 显示当前命令的下一条历史命令
ctrl+r: 搜索历史命令，随着输入会显示历史命令中的一条匹配命令，Enter键执行匹配命令；ESC键在命令行显示而不执行匹配命令。
ctrl+g: 从历史搜索模式（Ctrl – r）退出。
```



## 控制命令

```
ctrl+l: 清除屏幕，然后，在最上面重新显示目前光标所在的这一行的内容。
ctrl+o: 执行当前命令，并选择上一条命令。
ctrl+s: 阻止屏幕输出
ctrl+q: 允许屏幕输出
ctrl+c: 终止命令
ctrl+z: 挂起命令
ctrl+i: 等同于按制表符<TAB>键，命令自动填充
ctrl+m: 等同于回车键
ctrl+o: 等同于回车键
```



## Bang(!)命令

```
!! ：执行上一条命令。
^foo^bar ：把上一条命令里的foo替换为bar，并执行。
!wget ：执行最近的以wget开头的命令。
!wget:p ：仅打印最近的以wget开头的命令，不执行。
!$ ：上一条命令的最后一个参数， 与 Alt - . 和 $_ 相同。
!* ：上一条命令的所有参数
!*:p ：打印上一条命令是所有参数，也即 !*的内容。
^abc ：删除上一条命令中的abc。
^foo^bar ：将上一条命令中的 foo 替换为 bar
^foo^bar^ ：将上一条命令中的 foo 替换为 bar
!-n ：执行前n条命令，执行上一条命令： !-1， 执行前5条命令的格式是： !-5
```



## 其他命令

```

ctrl-V 使下一个特殊字符可以插入在当前位置, 如CTRL-V <TAB> 可以在当前位置插入一个<TAB>字符, 其ASCII是9, 否则一般情况下按<TAB>结果是命令补齐

ctrl-S 暂时冻结当前shell的输入
ctrl-Q 解冻



基本功:
用上下键看命令的历史
左右键区修改内容
tab补齐命令名字或者目录，文件名字，不是唯一的多按2次，会出来列表
!ls 重复运行最后一条以’ls’开头的命令，如果先ls -l 然后ls -lcrt，那么!ls，相当于ls -lcrt
 
 
ls abc.txt
vi !$
 
 
第二行的vi !$相当于vi abc.txt，!$等于上一个命令的参数， ‘$’ 是根据上下文来说的最后一行，列等。
 
 

```
